## Overview
Lejasrozes Forest Spa is dedicated to providing a peaceful retreat in nature, focusing on wellness and relaxation. The website serves as an informational hub for potential visitors and current clients.

## Features
- Elegant design that reflects the natural surroundings.
- Information on services offered including massages, nature walks, and wellness programs.
- Booking system for appointments.
- Contact information and location details.

## Installation
Instructions on how to set up the project locally:
1. Clone the repository:
   ```bash
   git clone https://github.com/lejasrozes/Nak-o-pie-dabas.git
   ```
2. Navigate into the project directory:
   ```bash
   cd Nak-o-pie-dabas
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage
To run the project, use the following command:
```bash
yarn start
```
